React Clicky-Game

Description

The application render different images to the screen. Each image listen for click events. The application keep track of the user's score. The user's score is incremented when clicking an image for the first time. The user's score is reset to 0 if they click the same image more than once. Every time an image is clicked, the images rendered to the page is shuffled in a random order. Once the user's score is reset after an incorrect guess, the game restart.

Technologies Used: ReactJS, MongoDB, Express Handlebars

Link to the Clicky-Game https://pure-everglades-30059.herokuapp.com/
